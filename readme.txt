Additional Script for Header and Footer

== Description ==

The plugin gives you flexibility for additing count down in you page. 

== Screenshots ==

1. Option of Additional Script in theme's customizer 
2. Editor panel for Head JS and Footer JS

== Frequently Asked Questions ==

= Who can use Additional Count Down plugin? =

Basically this plugin is for everyone who needs to add count down on their site.

= How Additional Count Down is useful? =

This plugin provides you the privileges for adding count down on site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

3. Activate the plugin through the 'Plugins' screen in WordPress

4. Find this Count Down menu
